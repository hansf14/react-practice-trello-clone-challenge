- **Features**
  - **Default 4 categories**
    - **"All" (Unremovable)**
    - **"To Do"**
    - **"Doing"**
    - **"Done"**
  - **Category CRUD**
    - **i.e. Also supports custom category**
    - **Supports multi-line category**
  - **Category search**
  - **To-do CRUD**
    - **Supports multi-line to-do text**
  - **Persistency (refresh, reopen)**
  - **Resizable multi-line to-do text input**
